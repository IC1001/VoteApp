## React投票网站

技术栈：React(使用create-react-app快速开发、canvas) + Ant design + Node.js(Express框架) + MongoDB

## 项目启动
### 启动MongoDB，创建react数据库 `use react`
### `cd react`
  react文件夹在终端打开
### `npm install`
  安装所需依赖
### `npm run build`
  生成前端打包好的页面
### `cd server`
### `npm install`
### `node app.js`或`nodemon app.js`
  启动项目，3000端口浏览，完成
